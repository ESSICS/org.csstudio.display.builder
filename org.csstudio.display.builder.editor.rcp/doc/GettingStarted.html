Getting Started

Downloading Display Builder Examples

In CS-Studio, menu CS-Studio > Utilities > Install Samples; select Display Builder and click OK. The examples will be in a project folder named Display Builder.

Run 01_main.bob in Display Runtime for links to information on many widgets and features of Display Builder, or open the files in Display Editor to observe widget properties and features of the editor interface.

Opening the Display Editor Perspective

The Display Editor perspective offers the tools and settings needed to edit Display Builder (*.bob) files. From CS-Studio, the Display Editor perspective can be opened from the menu bar: Window > Open Perspective > Other...; select Display Editor.

Alternatively, right-click on a *.bob or *.opi file in the Navigator and select Open With... > Display Editor (or Other > Display Editor). Then right-click in the central Editor part and select Open Editor Perspective. If you are already in that perspective, a dialog will appear that confirms if you want to reset the perspective to its default layout.

Creating a New Display

From within the Display Editor perspective, right-click on a project in the Navigator and select New > New Display; this will open the New Display wizard. (Alternatively, select Other... or go to File > New..., and under Display Editor, select the New Display wizard, then click Next.) From the wizard, you can choose a container and name for the new display file.

Converting a BOY OPI File

With Display Builder, opening and converting old BOY OPI files is simple. If you right-click on an *.opi file and Open With... > Display Editor, the file is automatically converted to the Display Builder format, substituting the closest widget or property in cases where Display Builder has consolidated. In most cases it can be saved directly to a new *.bob file with zero or few changes necessary. Be aware that any scripts associated with the display might need to be updated. Scripts are the most likely cause of failures in converted files.

BOY OPI files can also be run directly in Display Runtime without conversion.

Display Runtime

Display Runtime is the perspective in which displays are executed. To run a display, right-click on the file to Open With... > Display Runtime, or, from the editor, click the Run icon on the toolbar (a green and white arrow).
