Widgets

Widgets are the components of Display Builder displays. They can be configured by editing their properties (most often size, location, and PV name).

Key properties common to most widgets include:

  • type: Type of widget (e.g. label, text_update). Each widget has both a localizable (translatable) type name and an internal type name used to identify the widget type in display files.
  • name: Used to identify the widget; unique within the display.
  • x, y: Position of widget.
  • width, height: Size of widget.
  • foreground color (foreground_color): The color of text associated with the widget; or, if there is no text, of "foreground" display features like the border of a Group widget.
  • actions: attach Actions that do things like open displays or run scripts.
  • scripts: attach Scripts that dynamically control widget appearance and behavior.
  • rules: attach Rules that dynamically assign widget property values based on conditional expressions.
